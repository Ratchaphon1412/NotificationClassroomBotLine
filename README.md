# LINE Notification Classroom Bot

---

## fist step

- create python environments

for MacOS:

```bash
    python3 -m venv env
    source env/bin/activate
```

for Windows:

```cmd
    python3 -m venv env
    env\Scripts\activate.bat
```

## install requirements

```py
python3 install -r requirements.txt

```
